ตารางหน่วยการเรียนรู้และสมรรถนะประจำหน่วย

หน่วยการเรียนรู้ที่	สมรรถนะด้าน

หน่วยการเรียนรู้ที่ 1: ความรู้เกี่ยวกับการพัฒนาเว็บ และโครงสร้างภาษา PHP	
	สมรรถนะด้านความรู้
		1. มีความรู้พื้นฐานเกี่ยวกับเทคโนโลยีเว็บ
		2. มีความรู้เกี่ยวกับภาษา PHP
	สมรรถนะด้านทักษะ
		1. สามารถติดตั้งและใช้งานโปรแกรมที่เกี่ยวข้องกับการพัฒนาเว็บ
		2. สามารถเขียนภาษา PHP เบื้องต้น
	สมรรถนะด้านคุณลักษณะที่พึงประสงค์
		1. มีความสนใจและใฝ่รู้ในด้านการพัฒนาเว็บ
		2. มีความอดทนและแก้ไขปัญหา

หน่วยการเรียนรู้ที่ 2: การเขียนโปรแกรม PHP เบื้องต้น	
	สมรรถนะด้านความรู้
		1. มีความรู้เกี่ยวกับหลักการพื้นฐานของการเขียนโปรแกรม PHP
		2. มีความรู้เกี่ยวกับชนิดข้อมูลและตัวดำเนินการ
	สมรรถนะด้านทักษะ
		1. สามารถเขียนโค้ด PHP เพื่อแสดงผลข้อมูลพื้นฐาน
		2. สามารถใช้ตัวแปรและตัวดำเนินการใน PHP
	สมรรถนะด้านคุณลักษณะที่พึงประสงค์
		1. มีความละเอียดรอบคอบในการเขียนโค้ด
		2. มีความคิดสร้างสรรค์ในการแก้ไขปัญหา

หน่วยการเรียนรู้ที่ 3: ฟังก์ชันและการจัดการข้อมูลใน PHP	
	สมรรถนะด้านความรู้
		1. มีความรู้เกี่ยวกับฟังก์ชันใน PHP
		2. มีความรู้เกี่ยวกับการจัดการข้อมูลแบบ Array และ String
	สมรรถนะด้านทักษะ
		1. สามารถสร้างและใช้งานฟังก์ชันใน PHP
		2. สามารถจัดการข้อมูลแบบ Array และ String ใน PHP
	สมรรถนะด้านคุณลักษณะที่พึงประสงค์
		1. มีความสามารถในการวิเคราะห์และออกแบบโค้ด
		2. มีความสามารถในการทำงานร่วมกับผู้อื่น

หน่วยการเรียนรู้ที่ 4: การจัดการฟอร์มใน PHP

		2. สามารถรับค่าจากฟอร์มและประมวลผลใน PHP
	สมรรถนะด้านคุณลักษณะที่พึงประสงค์

		2. มีความสามารถในการแก้ไขปัญหาที่เกิดจากการใช้งานฟอร์ม

หน่วยการเรียนรู้ที่ 5: การทำงานกับคุกกี้และเซสชั่น	
	สมรรถนะด้านความรู้

		2. มีความรู้เกี่ยวกับการจัดการคุกกี้และเซสชั่นใน PHP

		1. สามารถสร้างและใช้งานคุกกี้
		2. สามารถสร้างและใช้งานเซสชั่น

		2. มีความสามารถในการแก้ไขปัญหาที่เกี่ยวข้องกับคุกกี้และเซสชั่น
