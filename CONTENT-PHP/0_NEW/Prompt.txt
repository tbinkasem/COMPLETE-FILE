Prompt 1

ในฐานะที่ท่านเป็นครูผู้สอนในสาขาวิชาเทคโนโลยีสารสนเทศ ขอให้ท่านแบ่งหน่วยการเรียน รายวิชา การพัฒนาเว็บด้วยภาษา PHP 
ตามคำอธิบายรายวิชา โดยอิงสมรรถนะรายวิชาด้วย ซึ่งในรายวิชานี้ใน 1 สัปดาห์  จะประกอบด้วย ทฤษฎี 1 ชั่วโมง ปฏิบัติ 4 ชั่วโมง 
ทำการเรียนจำนวนทั้งสิ้น 18 สัปดาห์ โดยใช้ข้อมูลต่อไปนี้ 

จุดประสงค์รายวิชา
	1. มีความรู้ ความเข้าใจ เกี่ยวกับการโปรแกรมเว็บด้วยภาษา PHP
	2. ปฏิบัติการโปรแกรมเว็บด้วยภาษา PHP
	3. มีกิจนิสัยในการทำงานด้วยความประณีต รอบคอบและปลอดภัย ตระหนักถึงคุณภาพของงานและมีจริยธรรมในงานอาชีพ

สมรรถนะรายวิชา
	1. การออกแบบโปรแกรมเว็บด้วยภาษา PHP ให้เหมาะสมกับงาน
	2. ปฏิบัติการโปรแกรมเว็บด้วยภาษา PHP ตามที่ได้ออกแบบไว้
	3. ทดสอบโปรแกรมเว็บที่สร้างขึ้น

คำอธิบายรายวิชา
	ศึกษาและปฏิบัติเกี่ยวกับหลักการสื่อสารข้อมูลบนอินเทอร์เน็ต รูปแบบ Client Server การทำงานของ Web Server 
การติดตั้งเครื่องมือและโปรแกรมที่ใช้ในการพัฒนาเว็บด้วยภาษา PHP ลำดับขั้นตอนในการพัฒนาเว็บ ข้อมูล ตัวแปร ตัวดำเนินการ 
การควบคุมคำสั่ง ฟังก์ชั่น การจัดการแฟ้มข้อมูล คุกกี้ เซสชั่น การรับส่งข้อมูลผ่านฟอร์ม การอัปโหลดไฟล์ 
การเชื่อมต่อและใช้งานระบบฐานข้อมูล การสร้างเว็บไซต์เพื่อประยุกต์ใช้งานในหน่วยงานหรือภาคธุรกิจ

อยากให้ท่านแบ่งหน่วยการเรียนรู้ จำนวน 10 หน่วย โดยข้อมูลหน่วยการเรียนรู้แต่ละหน่วยประกอบด้วย 
ชื่อหน่วยการเรียนรู้ จำนวนชั่วโมงแต่ละหน่วย และสัปดาห์ 
นำเสนอในรูปแบบตาราง ชื่อ ตารางหน่วยการเรียนรู้ ประกอบด้วย คอลัมน์ที่ 1 เป็น หน่วยที่ คอลัมน์ที่ 2 เป็น ชื่อหน่วยการเรียนรู้ 
คอลัมน์ที่ 3 เป็น จำนวนชั่วโมงของหน่วยการเรียนรู้ และ คอลัมน์ที่ 4 เป็น สัปดาห์ที่ ด้วยครับ


Prompt 2

ผมอยากให้นำข้อมูลจากตารางหน่วยการเรียนรู้ โดยนำมาสร้างตารางใหม่ ชื่อว่า ตารางหน่วยการเรียนรู้และสมรรถนะประจำหน่วย 
โดยในตารางจะประกอบไปด้วย 2 คอลัมน์ คือ หน่วยการเรียนรู้ และ สมรรถนะ 
ในส่วนของสมรรถนะ โดยนำข้อมูลจากคำอธิบายรายวิชา ซึ่งแบ่งออกเป็น  3 ด้าน 
คือ ความรู้ ทักษะ คุณลักษณะที่พึงประสงค์ โดยให้แสดงรายการทั้ง 3 ด้านออกเป็นเลขลำดับ
